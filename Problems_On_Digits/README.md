# Digit-Based Programming Problems in C

This directory contains a collection of C programs focused on digit manipulation, number theory, and mathematical logic. Each program follows a strict documentation style including algorithms and test cases.

## 📂 Directory Structure

| File Name | Description | Logic Used |
| `CheckStrong.c` | Checks if a number is a Strong Number | Factorial Lookup Table |
| `CheckTrioMorphic.c` | Verifies Triomorphic property | Cube & Digit matching |
| `FibonacciRecur.c` | Generates Fibonacci series | Static Recursion |
| `FindMaxDigit.c` | Finds the largest digit | Iterative Comparison |
| `ReverseNumber.c` | Reverses an integer | Modulo & Multiplicative Shift |

## 🚀 How to Run

To compile and run any program, use a C compiler like `gcc`:

1. Open your terminal/command prompt.
2. Navigate to this directory.
3. Run the following commands:
   ```bash
   gcc FileName.c -o MyExe
   ./MyExe
